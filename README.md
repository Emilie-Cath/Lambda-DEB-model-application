This is the repository for lambda-DEB model.
It is linked to the (future) publication Estimating isotopic turnover rates using DEB theory.
Co-authors are Emilie Cathelin, Sébastien Lefebvre and Carolina Giraldo


This git-hub coutains 5 files to run it.
#-----
DEB-paramV6.csv is the DEB database with all parameters of interest for all described species easy to open in R.
#-----
dataset_commun_DEB_add_temp.csv : is the excel containing dataset 1 and 3
half_life_review.csv : is the excel contaning dataset 2
#-----
functions_lambda_DEB.R is a code with all functions describing lambda-DEB, Thomas and Crowther and Vander Zanden relationships, DEB fluxes and respiration function
#-----
lambda_DEB_model.R is the main code to run for generating article figure 4 to 9.
